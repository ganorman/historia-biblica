# 📖 La Gran Historia Bíblica

Línea de tiempo interactiva que recorre toda la historia bíblica desde la **Creación (~4000 a.C.)** hasta el **Nuevo Testamento (~100 d.C.)**.

## 🌐 Ver en vivo

👉 **[ganorman.github.io/historia-biblica](https://ganorman.github.io/historia-biblica/)**

## ✨ Características

- **Línea de tiempo completa** con 11 eras históricas
- **106 personajes bíblicos** con biografías, versículos y árbol familiar
- **65 eventos clave** con contexto histórico y referencias
- **Filtros por tipo** — Patriarcas, Reyes, Profetas, Jueces, Apóstoles, Mujeres
- **Saltos rápidos** a los momentos más importantes
- **Selector de año** con rango -4000 a +100
- Diseño profesional oscuro con identidad de marca personal

## 🛠️ Tecnología

Aplicación de archivo único — HTML + CSS + JavaScript puro, sin frameworks ni dependencias de build.
